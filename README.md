# 🎉 FestiveWishBot

A professional Telegram bot for creating festive-themed phishing campaigns with camera and location permission requests. Inspired by CamPhish/Zphisher but with a festive twist.

## ✨ Features

- **7 Professional Festive Templates** (Eid, Holi, Easter, Diwali, Christmas, New Year)
- **Automatic Tunnel Creation** (ngrok/cloudflared)
- **Camera & Location Permission Requests**
- **Real-time Victim Notifications**
- **Device Fingerprinting**
- **IP Geolocation**
- **Professional Console Output**
- **Railway Deployable**

## 📋 Prerequisites

- Python 3.8 or higher
- Telegram Bot Token (from @BotFather)
- ngrok account (optional, for custom domain)
